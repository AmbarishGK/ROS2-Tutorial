# Tutorial 10: RQT Tools and Debugging

RQT provides GUI-based visualization and debugging tools.

## 10.1 Common RQT Tools
- `rqt_graph`: visualize node/topic connections.
- `rqt_plot`: plot topic values.
- `rqt_console`: view log messages.

## 10.2 Example
```bash
rqt_graph

```
